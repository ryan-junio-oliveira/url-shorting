<h1>Projeto Encurtador de URL</h1>
<p>Este é um projeto de encurtador de URLs desenvolvido em Laravel, permitindo criar URLs curtas, redirecioná-las para a URL original e consultar estatísticas detalhadas de cada URL encurtada.</p>
